# workbench

This repository contains a reusable cyclone separator CFD starter project for ANSYS Workbench and Fluent.

## Included content

- `cyclone_separator_case/`: a case package with geometry dimensions, Fluent setup notes, and a schematic
- `skills/cyclone-separator-simulation/`: a Codex skill for reusing the same cyclone separator simulation workflow later
